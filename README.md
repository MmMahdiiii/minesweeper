# minesweeper
**this game is desgined in 3 levels :**  
Level 1 : square(5 * 5) & 4 bombs & time : 200 s   
Level 2 : square(12 * 12) & 28 bombs & time : 800 s   
Level 3 : square(20 * 20) & 96 bombs & time : 2000 s   
_and also you can play this game on custom mode_
